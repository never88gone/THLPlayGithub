# 隐私政策 (Privacy Policy)

**生效日期：2026年4月27日**

糖葫芦享屏（以下简称“本应用”）由糖葫芦开发团队（以下简称“我们”）开发并运营。我们非常重视您的隐私，本隐私政策旨在向您说明本应用如何处理信息。

### 1. 核心隐私原则
**本应用致力于保护用户隐私。最重要的一点是：当前应用不具备任何后端服务器，您的所有数据和操作均在本地设备或局域网内完成。**

### 2. 信息收集与使用
我们承诺：
- **不收集个人身份信息**：我们不收集您的姓名、电话号码、电子邮件地址、位置信息或任何其他可识别个人身份的信息。
- **不收集设备数据**：我们不收集您的设备唯一标识符（UDID）、IMEI 或其他硬件追踪信息。
- **无数据上传**：由于本应用没有服务器，您的任何使用记录、音视频内容或配置信息均不会被上传至云端或任何第三方。

### 3. 权限说明
为了实现 AirPlay 接收功能，本应用需要申请以下权限：
- **本地网络 (Local Network)**：这是核心功能所必需的。用于在您的家庭或办公局域网内发现投屏设备并建立连接。
- **蓝牙 (Bluetooth)**：用于辅助发现附近的 Apple 设备（AWDL 协议），以提高连接成功率。
- **音频与显示**：本应用仅在投屏期间实时渲染接收到的音频和视频流，不会对流内容进行录制、存储或分析。

### 4. 第三方服务
本应用不包含任何第三方 SDK（如广告插件、数据统计分析插件或社交分享插件）。因此，不存在任何与第三方共享信息的情况。

### 5. 数据安全
由于所有数据仅在您的本地局域网内传输，且应用本身不存储任何音视频数据，因此在物理层面上避免了远程泄露的风险。我们建议您在受信任的加密 Wi-Fi 环境下使用本应用。

### 6. 儿童隐私
本应用不收集任何个人信息，因此也适用于儿童。我们不会有意收集 13 岁以下儿童的任何信息。

### 7. 政策修订
我们可能会适时更新本隐私政策。任何修订都将通过更新本页面来通知。

### 8. 联系我们
如果您对本隐私政策有任何疑问，请通过 GitHub 仓库提交 Issue 与我们联系。
